[ 스프링 과제05 ]
task 패키지 폴더에서 진행
주제 : 게시판 RestController 만들기
    1. AppStart 클래스 생성
    2. BoardController 클래스 생성
    3. BoardService 클래스
    4. BoardMapper 인터페이스 [ 임의의테이블명 생성 : taskboard ]
     - xml파일 없이 @SQL 어노테이션 이용하여 SQL 매핑한다.
    5. BoardDto 클래스 생성
    6. fetch 대신에 axios(동기vs비동기 선택) 사용한다.
    7. resource -> static -> task 폴더 생성하여 index.html 하나의 html만 사용한다.

        1. 글쓰기           POST       "/spring/task/board"
            요청자료 : { "title" : "" , "content" : "" }
            응답자료 : true 또는 false

        2. 전체 글 조회      GET        "/spring/task/board"
            요청자료 : x
            응답자료 : [ { "bno": "" , "title" : "" , "content" : "" } ,
                        { "bno": "" , "title" : "" , "content" : "" }  ]

        3. 개별 글 조회      GET        "/spring/task/board/view"
            요청자료 : ?bno=1
            응답자료 : { "bno": "" , "title" : "" , "content" : "" }

        4. 개글 글 수정      PUT        "/spring/task/board"
            요청자료 : { "bno": "" , "title" : "" , "content" : "" }
            응답자료 : true 또는 false

        5. 개별 글 삭제      DELETE     "/spring/task/board"
            요청자료 : ?bno=1
            응답자료 : true 또는 false
